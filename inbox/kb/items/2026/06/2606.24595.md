## Summary
MEMPROBE presents an evaluation framework for assessing whether LLM agents actually retain and understand user information over extended interactions. The key innovation is shifting from task-completion metrics to direct measurement of what the agent learns about users—by reconstructing structured user state from memory after interactions. The paper includes a benchmark with 50 simulated users with 31 hidden personal dimensions each.
