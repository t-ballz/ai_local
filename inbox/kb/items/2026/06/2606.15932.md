## Summary
A structured survey on Multimodal Code Intelligence—systems that generate, edit, or reason with code based on visual inputs (screenshots, charts, videos, UI states). Categorizes the field by code role and organizes tasks into four domains: GUI code generation, scientific visualization, structured graphics, and frontier agentic applications. Emphasizes verification-centered research directions.
