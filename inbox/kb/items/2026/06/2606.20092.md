## Summary
EventVLA is a vision-language-action (VLA) policy framework that solves memory limitations in long-horizon robotic manipulation by using sparse visual evidence memory. It combines foundational visual anchors with a Keyframe Evidence Memory module that predicts which observations are task-critical and preserves them before they become unobservable, achieving +40% improvement over existing memory-augmented VLAs on both simulation and real-world tasks.
