# Knowledge Base Tag Vocabulary

Tags are used to cross-link items and power the knowledge graph.
Each tag is `[a-z0-9-]+`. Choose 2-5 per item.
Propose a new tag with `?new-tag` prefix; a human will decide whether to adopt it.

## Topic

- `agents` — agent systems, tool use, agentic loops, scaffolding
- `robotics` — embodied AI, manipulation, locomotion, sim-to-real
- `vla` — vision-language-action models
- `world-models` — predictive environment simulators for planning
- `training` — fine-tuning, SFT, RLHF, data recipes, instruction tuning
- `rl` — reinforcement learning algorithms and applications
- `evaluation` — benchmarks, metrics, leaderboards, scoring
- `benchmark` — new benchmark datasets or evaluation suites
- `image-generation` — text-to-image, inpainting, style transfer
- `video-generation` — text-to-video, video prediction, animation
- `diffusion` — diffusion model architectures and techniques
- `moe` — mixture-of-experts architectures
- `attention` — attention mechanisms, variants, and optimizations
- `long-context` — extended context windows and retrieval
- `quantization` — model compression via quantization
- `inference` — inference optimization, serving, deployment, KV cache
- `embedding` — text/multimodal embeddings and rerankers
- `retrieval` — information retrieval, RAG, search
- `rag` — retrieval-augmented generation
- `reasoning` — chain-of-thought, math, logic, planning, verification
- `distillation` — knowledge distillation, model compression
- `multimodal` — multi-modal models (vision+language, audio, etc.)
- `ocr` — optical character recognition
- `asr` — automatic speech recognition
- `tts` — text-to-speech synthesis
- `datasets` — training data, data curation, synthetic data generation
- `prompt-optimization` — automated prompt tuning and optimization
- `code-generation` — code synthesis, programming agents, software engineering
- `safety` — alignment, safety, red-teaming, jailbreaks
- `gui-agents` — graphical user interface automation, mobile/desktop agents
- `3d` — 3D generation, novel view synthesis, point clouds, NeRF/Gaussian
- `pretraining` — pre-training methods, data schedules, scaling
- `architecture` — novel model architectures
- `efficiency` — compute efficiency, sparse methods, fast inference
- `music` — music generation, audio synthesis
- `biology` — biological AI, protein folding, drug discovery

## Type

- `model-release` — new open-weight model release
- `paper` — research paper or technical report
- `tool` — software tool, framework, or library
- `technique` — specific training/inference technique or method
- `tip` — practical tip, guide, or workflow
